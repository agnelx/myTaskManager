📅 Connect Outlook Calendar
One-time setup required. You need a free Azure App Registration to allow this app to read your Outlook calendar. This is safe — the app only gets read-only calendar access using your Microsoft account.
Step 1 — Register the app in Azure

Go to portal.azure.com → App registrations → New registration
Name: MyTaskManager · Account type: Accounts in any org + personal
Redirect URI: Single-page application (SPA) → paste the URL of this HTML file.

Step 2 — Add API permissions

In your app → API permissions → Add → Microsoft Graph → Delegated:
Add Calendars.Read and User.Read → click Grant admin consent.

Step 3 — Paste your Client ID below

Copy the Application (client) ID from the app overview page.
